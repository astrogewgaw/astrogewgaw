## what am i up to?

a lot of stuff! right now, i am building a fully-automated data-processing pipeline for searching periodic sources (such as pulsars) and transient sources (such as **f**ast **r**adio **b**ursts, or **FRB**s) in the radio regime. to make a pipeline, you require a lot of pipes (:wink:) so i am usually busy making a lot of submodules that will help with each step of the process. a culmination of some of these efforts is the **ptypes** package. **p**types means **p**ulsar data types, and it is a pure Python package meant to read/write and process pulsar data, such as time series, or frequency-time arrays. since Python is slow, i am writing C++ code for the actual data processing algorithms in a separate package: **pkernels**. if you have any ideas you would like to share, send me an email and we can talk :grin:!

i am also looking for research positions in **astrochemistry**, a.k.a. the study of molecules in space, so if you see something, say something :wink:! i am a chemistry undergrad *and* an electronics engineer, and i really wanted to enter a field of research that could combine all of these interests and skills of mine into one. **astrochemistry** is one of those fields, because it involves astronomy (a lot of which is in the radio regime, since a lot of molecules have detectable rotational spectra) *and* **chemistry** (through simulation of these reactions in space-like environments in the laboratory or on the computer). thanks to my weird choice of degrees and my crooked path in research so far, i ended up accumulating a lot of skills that i think make me a must-have asset for an interdisciplinary field like this one. so, if any prospective employers end up on this page, do consider hiring an oddball like me on your team. you will not be disappointed :smile:!
